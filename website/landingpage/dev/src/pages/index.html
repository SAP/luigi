Benefits
Build administration and business User Interfaces using Luigi and discover its benefits.

  • Modular

    Decompose your monolithic web applications into a set of consistent and clear UI modules reflecting their underlying backend modularity.

  • Extensible

    Seamlessly integrate UI modules from external systems and build applications that can be extended with additional functionality from your customer or a 3rd party vendor in a secure way.

  • Scalable

    Divide and distribute the development of end-to-end features to an arbitrary number of teams, who can then independently and rapidly develop, deploy, maintain and operate their solutions.

  • Technology agnostic

    Technologies change often. You never know, if a technology you've chosen today, will be outdated in a year. Luigi is technology agnostic and allows you to react to new trends quickly preventing technology lock-ins.

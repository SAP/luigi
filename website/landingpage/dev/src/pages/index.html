Benefits
Build administration and business User Interfaces using Luigi and discover its benefits.

  • Modular

    Decompose your monolithic web applications with a fixed scope into a set of UI modules reflecting the underlying backend modularity.

  • Extensible

    Seamlessly integrate UI modules from external systems and build applications that can be extended with additional functionality from you customer or a 3rd party vendor in a secure way.

  • Scalable

    Distribute the development of end-to-end features to an arbitrary number of teams to develop, deploy, maintain and operate each module independently.

  • Agile

    Technologies change often. You never know, if a technology you've chosen today, will be outdated in a year. Luigi is technology agnostic and allows you to react to new trends quickly preventing technology lock-ins.

Navigation Synchronization

Summary

This page uses the Angular router to navigate between pages. The URL of the microfrontend is updated through the LuigiAutoRoutingService of the LuigiAngularSupportModule, which utilized router events to update the URL on Luigi Core side by using the LuigiClient.linkManager().

From Context

Uses fromContext: true in the Angular route in order to make Luigi navigate to a path relative to the closest navigation context.
